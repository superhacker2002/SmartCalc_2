Инструкция по использованию Smart_Calc_v1.0

Установка:

  1. Вызвать цель "make install" из папки с  source-файлами.
  2. Дождаться установки *.app.

Удаление:

  1. Вызвать цель "make uninstall".
  2. Дождаться удаления  *.app.

Запуск и использование:

  1. Для запуска приложения открыть исполняемый файл Smart_calc.app, либо через терминал командой "open Smart_calc.app"
  2. Для вычисления: ввести выражение в поле с клавиатуры, либо кнопками и нажать на кнопку "=".
  3. Для построения графика: ввести выражение, перевести калькулятор в режим построения графика,  выбрав соответствующую "radio button", нажать  на кнопку "Построить график".
  4. Для запуска кредитного калькулятора нажать на кнопку "Кредитный калькулятор", ввести данные в открывшемся окне, нажать на кнопку "Посчитать".

Запуск тестов:

  1. Для запуска тестов подсчета вызвать цель "make test_src".
  2. Для создания html страницы с данными о покрытии функций тестами: вызвать цель "make gcov_report".
